# Homelab Fetch
A fetch tool that uses SSH to display information about servers in your homelab

![](img/screenshot.png)

# Usage

```
$ hlfetch --help
usage: hlfetch [-h] -t TARGET

A fetch tool that uses SSH to display information about servers in your homelab

options:
  -h, --help            show this help message and exit
  -t TARGET, --target TARGET
                        A target to fetch (username@host)
```
